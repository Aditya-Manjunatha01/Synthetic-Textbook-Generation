### 2.1.3 Exercise: Proving the Principle of Maximum Entropy for an Isolated System

Having established the Clausius inequality and the property of entropy as a state function, we can now formally derive the Principle of Maximum Entropy. This principle serves as the cornerstone of the Second Law of Thermodynamics, providing a criterion for the direction of spontaneous change and the definition of thermodynamic equilibrium.

Consider an isolated system. By definition, an isolated system is one that exchanges neither matter nor energy with its surroundings. Consequently, there is no heat exchange ($\delta Q = 0$) and no work performed ($\delta W = 0$). From the general relation for entropy change derived in Subsection 2.1.1, we have:

$$dS \ge \frac{\delta Q}{T}$$

For an isolated system, where $\delta Q = 0$, this inequality simplifies to:

$$dS \ge 0$$

This result indicates that the entropy of an isolated system can either remain constant (during a reversible process) or increase (during an irreversible process), but it can never decrease. Thus, the entropy $S(t)$ is a monotonically non-decreasing function of time.

To determine the state of equilibrium, we define thermodynamic equilibrium as a state in which the macroscopic variables of the system are time-independent. At this state, no further spontaneous macroscopic changes occur, implying that the entropy has ceased to increase:

$$dS = 0$$

We can prove that this equilibrium state corresponds to a maximum of entropy using a proof by contradiction. Assume that a system is in equilibrium at a state characterized by entropy $S_{eq}$, but that there exists some other accessible state with a higher entropy $S_{max} > S_{eq}$ (consistent with the constraints of isolation, such as fixed internal energy $U$ and volume $V$). 

If the system is truly in equilibrium, it must remain in state $S_{eq}$ indefinitely. However, if a state $S_{max}$ is accessible, the transition from $S_{eq}$ to $S_{max}$ would constitute a spontaneous process. According to the Clausius inequality for an isolated system, such a process is permissible because $dS = S_{max} - S_{eq} > 0$. Since the system will spontaneously evolve toward any accessible state of higher entropy, it cannot be said to be in equilibrium until no such higher-entropy state is accessible.

Therefore, the system reaches thermodynamic equilibrium if and only if its entropy has reached the maximum value possible given the external constraints. We can express this condition as:

$$S_{eq} = S_{max}$$

In summary, for any spontaneous process in an isolated system, the entropy increases until it reaches a maximum value, at which point the system is in equilibrium. This behavior is typically visualized as an asymptotic approach to a maximum value $S_{eq}$ over time, where the rate of increase $dS/dt$ vanishes as the system nears the equilibrium state.
