

<<CONCEPT: Helmholtz Free Energy>>
name        : Helmholtz Free Energy
status      : proven
location    : 2.3.1
depends_on  : []
statement   : The portion of a system's internal energy available to do work at constant temperature and volume.
<</CONCEPT>>

<<CONCEPT: Gibbs Free Energy>>
name        : Gibbs Free Energy
status      : proven
location    : 2.3.1
depends_on  : []
statement   : The maximum non-expansion work obtainable from a system at constant temperature and pressure.
<</CONCEPT>>

<<CONCEPT: Spontaneity Criteria>>
name        : Spontaneity Criteria
status      : proven
location    : 2.3.1
depends_on  : []
statement   : A process is spontaneous if it leads to a decrease in Helmholtz free energy (at constant T, V) or Gibbs free energy (at constant T, P).
<</CONCEPT>>

<<CONCEPT: Gibbs Spontaneity Criterion>>
name        : Gibbs Spontaneity Criterion
status      : proven
location    : 2.3.2
depends_on  : [G]
statement   : At constant T and P, a process is spontaneous if \Delta G < 0, at equilibrium if \Delta G = 0, and non-spontaneous if \Delta G > 0.
<</CONCEPT>>

<<CONCEPT: Gibbs-Helmholtz Relation>>
name        : Gibbs-Helmholtz Relation
status      : introduced
location    : 2.3.2
depends_on  : [G]
statement   : The change in Gibbs free energy is given by \Delta G = \Delta H - T\Delta S.
<</CONCEPT>>

<<CONCEPT: Gibbs-Helmholtz Equation>>
name        : Gibbs-Helmholtz Equation
status      : proven
location    : 2.3.3
depends_on  : []
statement   : Relates the temperature dependence of the Gibbs free energy to the enthalpy of the system via (\partial(G/T)/\partial T)_P = -H/T^2.
<</CONCEPT>>

<<CONCEPT: van 't Hoff Equation>>
name        : van 't Hoff Equation
status      : proven
location    : 2.3.3
depends_on  : []
statement   : Describes the temperature dependence of the equilibrium constant K through the relation d(ln K)/dT = \Delta H^\circ / (RT^2).
<</CONCEPT>>
