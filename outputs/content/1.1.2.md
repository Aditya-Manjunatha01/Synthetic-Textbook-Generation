To understand the practical application of state variables and the geometry of the state space, it is useful to consider a physical system and determine the minimum number of variables required to specify its condition uniquely. This minimum number is known as the degrees of freedom, denoted by $\text{f}$.

Consider a simple thermodynamic system: a vertical cylinder containing a fixed amount of a pure gas, capped by a frictionless piston. The state of this gas can be described by various properties, such as the pressure $P$, the volume $V$, the temperature $T$, and the total internal energy $U$. While we can measure many such properties, they are not all independent. The degrees of freedom represent the number of independent intensive properties that must be specified to completely define the state of the system.

### Worked Example: Degrees of Freedom and Variable Independence

We will analyze a closed system consisting of a single phase of a pure substance.

**Part A: Determining the Degrees of Freedom**

To find the degrees of freedom, we follow a systematic counting procedure. First, we identify the total number of state variables $\text{N}$ relevant to the system. For a simple compressible substance, we typically consider the pressure $P$, temperature $T$, and volume $V$. However, volume is an extensive property. To ensure we are analyzing the state independent of the system's size, we convert it to the specific volume $v$, defined as:

$$v = \frac{V}{m}$$

where $m$ is the mass of the substance. By using $v$, we ensure all our variables are intensive. Thus, we have a set of $\text{N} = 3$ intensive variables: $\{P, T, v\}$.

Next, we identify the number of independent constraints $\text{R}$ acting on these variables. For a pure substance in a single phase, the state is governed by an equation of state—a functional relationship that links these variables, such as the ideal gas law $Pv = RT$. This equation constitutes one constraint ($\text{R} = 1$).

The degrees of freedom $\text{f}$ are calculated as the difference between the number of variables and the number of constraints:

$$\text{f} = \text{N} - \text{R}$$

Substituting our values:

$$\text{f} = 3 - 1 = 2$$

This result indicates that specifying any two of the three variables (e.g., $P$ and $T$) automatically determines the third ($v$) via the equation of state.

**Part B: Testing for Independence**

A set of variables is independent if no variable in the set can be expressed as a function of the others. If a functional relationship exists, the variables are dependent.

Suppose we are given the set $\{P, V, T\}$ for a closed system with a fixed mass $m$. We check for independence by looking for a relationship of the form:

$$f(P, V, T) = 0$$

For an ideal gas, this relationship is:

$$PV - nRT = 0$$

Because this equation exists, the variables $P, V,$ and $T$ are not independent. If we know $P$ and $T$, the value of $V$ is strictly constrained to:

$$V = \frac{nRT}{P}$$

Consequently, the dimensionality of the state space is reduced. While we might visualize the system in a 3D space with axes $P, V,$ and $T$, the actual states of the system are restricted to a 2D surface within that space. This is why thermodynamic processes are frequently represented on 2D projections, such as $P$-$V$ (pressure-volume) or $T$-$s$ (temperature-entropy) diagrams. A single point on a $P$-$V$ diagram uniquely identifies the state of the system because the third variable, $T$, is implicitly determined by the surface of the equation of state.
